#Ai-Camera
WIP
This project uses a yolo detection model trained on coco dataset, as well as a VLM the VLM I choose is Moondream its a small model that runs fast and my goal was to make it as instant or real time as possible for anyone that may want to wear the camera and or Install for their home use.

I will be adding some changes to this script such as video storage for anyone that uses as a security use,Adding multi camera support if needed.
